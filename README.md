# AstroTalks
Developed a website which shows information of various zodiac signs alongside a numerology calculator.
This application fetched data of daily horoscopes using POST request from an open API. 
(Technologies used: Bootstrap5, Html, Css, jQuery; Languages used: JavaScript; IDE-VScode)
